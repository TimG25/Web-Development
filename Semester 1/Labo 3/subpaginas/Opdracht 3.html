Elke accordion section bestaat uit een titel en een div-tag waar de tekst in staat die ook verborgen kan worden.
In het script zie ik een functie waarvan ik denk dat deze de wijziging uitvoert, deze noemt accordion.
Wat me opviel binnen de h3 class was dat de parameters aria-selected en aria-expanded veranderde.
Ook binnen de div class veranderde er parameters zoals aria-hidden.
